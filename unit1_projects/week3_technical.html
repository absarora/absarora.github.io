Currently, there are no web technologies that do not use Javascript. For example AJAX and jQuery are the most common web technologies used to produce a dynamic web design and web development.

AJAX stands for Asynchronous Javascript and XML, where AJAX is a group of technologies interrelated to create dynamic web pages; while, jQuery is a Javascript library used for creating animations and many other dynamic features for the webpages.
