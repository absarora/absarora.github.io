Enumerable #map

Map, often confused with "Each", is a Ruby method used to iterate over an array where Map will return a new array put together by the results of the block while Each will return the original array.

For example:

Each

irb(main):> [1, 2, 3].each { | i | i + 2 }
=> [1, 2, 3]

Map

irb(main):> [1, 2, 3].map { | i | i + 2 }
=> [3, 4, 5]

There is another Ruby method, called "Collect" which is identical to "Map"; Collect, also, outputs a new array from the block results.

Collect

irb(main):> [1, 2, 3].collect { | i | i + 2 }
=> [3, 4, 5]
